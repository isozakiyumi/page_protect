iOS:選択できない（user-select:none;）もしiPadとかでCtrl+Aされたときとか、念の為（oncopy="return false;"）
Android:選択できない（user-select:none;）

画像：
iPhoneやPCは画像を長押しやD&Dでコピー/DLはできない。（pointer-events: none;）
Android chromeで画像長押しできなくなった。（pointer-events: none）

全般、念の為に oncopy="return false;"いれておくか。。
リンク：
ondragstart = () => { return false; }
これだけだとAndroidでリンク上で長押し→「リンクテキストをコピー」でコピーできた（oncopyでは抑止できない。。）。。）ので、コンテキストメニュー抑止（このページはbodyにつけてある）
